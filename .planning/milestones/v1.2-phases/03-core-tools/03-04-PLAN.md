  <how-to-verify>
打开 http://localhost:5010/editor 按以下步骤验证：

1. 铅笔绘制 (B)
   - 按 B 激活铅笔（右栏铅笔按钮高亮）
   - 在画布上拖拽 — 应该画出清晰的像素线段，无间隙
   - 按 Cmd+Z — 整条笔触被撤销（一步）

2. 橡皮擦除 (E)
   - 按 E 激活橡皮
   - 在有像素的地方擦拭 — 应显示棋盘格（完全透明）
   - 按 Cmd+Z — 橡皮操作被撤销

3. 油漆桶 (G)
   - 按 G 激活油漆桶
   - 点击画布上一个纯色区域 — 该区域被填充为当前前景色
   - 调整顶栏 Tolerance 为 50，再次点击 — 容差更大的区域被填充
   - 取消 Contiguous 勾选，点击 — 画布上所有相似颜色的像素都被填充（非连通）

4. 常驻调色盘
   - 左栏底部始终可见调色盘（滚动左栏也不消失）
   - 拖拽 HSL 色轮外圈 — Hex 和 RGB 输入同步更新；前景色 swatch 变色
   - 拖拽 SL 方块内部 — 颜色更新
   - 在 Hex 框输入 #ff0000，按 Enter — 变为红色，色轮/RGB 同步
   - 修改 R 值为 0 — 颜色更新，色轮/Hex 同步
   - 点击复制按钮 — 粘贴到任意地方确认 hex 值正确

5. 取色器
   - 点击取色器按钮（或按调色盘中的按钮）
   - 悬停画布 — 十字准星跟随像素网格；调色盘旁出现小色块预览
   - 点击任意有色像素 — 前景色更新为该像素颜色；工具恢复为之前的工具
   - 再次激活取色器，悬停透明区域（棋盘格）— 显示 "transparent"，色块消失
   - 点击透明像素 — 前景色不变
   - 激活取色器后按 Escape — 取消，工具恢复，颜色不变

6. 鼠标预览
   - 铅笔/橡皮工具悬停画布 — cursor-canvas 显示笔刷形状预览，跟随像素网格跳动
   - 设置 Size 为 5 — 预览变为 5px 大小

7. Pixel-Perfect 模式
   - 铅笔工具，勾选 Pixel-Perfect
   - 以 45° 对角线方向缓慢拖拽 — 应产生纯对角线像素，无 L 角多余像素

确认以上各点正常后，回复 "approved"。如有问题请描述具体现象。
