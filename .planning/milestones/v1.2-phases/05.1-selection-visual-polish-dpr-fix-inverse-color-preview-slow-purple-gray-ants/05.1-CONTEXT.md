<domain>
## Phase Boundary

三个紧急 Bug 修复，全部在 `editor.html` 内部，不涉及 Flask / 后端：

1. **DPR 双重应用修复**：`drawAnts()` 和 Marquee 拖拽预览里的 `save(); scale(dpr, dpr); ...; restore()` 要删掉——`initCanvases()` 已经永久应用了 DPR 缩放，再叠一次会让路径在 Retina (DPR=2) 上以 DPR²=4 倍渲染。
2. **反色蚂蚁渲染**：蚂蚁描边改用 `globalCompositeOperation = 'difference'` + 白色单描边，任何背景上都可见（黑底→白边，白底→黑边）。
3. **慢速紫灰色视觉症状**：DPR 修复后自动解决（4px 宽线条 + 双色叠加是视觉症状的根因）。

</domain>

<decisions>
## Implementation Decisions

### DPR 修复范围（两处）
- `drawAnts()` 函数（约第 1122 行）：删除 `save()` / `scale(dpr, dpr)` / `restore()` 包装层；`clearRect` 改为逻辑坐标 `clearRect(0, 0, EditorState.width, EditorState.height)`
- Marquee 拖拽预览（约第 2428 行 marquee onMove 内）：同样删除 `save()` / `scale(dpr, dpr)` / `restore()` 包装层；`clearRect` 同上

### 反色蚂蚁实现
- 采用 `globalCompositeOperation = 'difference'`
- 只需单次白色描边（不再需要白 + 黑双描边）
- 绘制前设置，绘制后重置为 `'source-over'`
- 动画 dash offset 逻辑不变

### 不改变的内容
- `rebuildAntsPath()` 的 Path2D 构建逻辑不变
- `scheduleAnts()` / RAF 循环结构不变
- `EditorState.selectionMask` 数据模型不变
- 选区工具（Marquee/Wand）的交互逻辑不变
- Phase 5 的任何其他功能不变

### Claude's Discretion
- `lineWidth` 在修复后应为 1（设备无关逻辑像素），由 DPR init-scale 处理 Retina 渲染
- `setLineDash([4, 4])` dash 长度可保持不变，视觉效果需在修复后目视验证

</decisions>

<specifics>
## Specific Ideas

**需要修改的具体代码位置（editor.html）：**

```javascript
// 修复前（drawAnts）：
function drawAnts() {
  if (!EditorState.selectionMask || !_antsPath) { antsRafId = null; return; }
  const dpr = window.devicePixelRatio || 1;
  selCtx.clearRect(0, 0, selCanvas.width, selCanvas.height);  // BUG: 设备像素
  selCtx.save();
  selCtx.scale(dpr, dpr);  // BUG: 双重 DPR
  selCtx.lineWidth = 1;
  selCtx.setLineDash([4, 4]);
  selCtx.strokeStyle = '#ffffff';
  selCtx.lineDashOffset = antsDashOffset;
  selCtx.stroke(_antsPath);
  selCtx.strokeStyle = '#000000';
  selCtx.lineDashOffset = antsDashOffset + 4;
  selCtx.stroke(_antsPath);
  selCtx.restore();
  ...
}

// 修复后（drawAnts）：
function drawAnts() {
  if (!EditorState.selectionMask || !_antsPath) { antsRafId = null; return; }
  selCtx.clearRect(0, 0, EditorState.width, EditorState.height);  // 逻辑像素
  selCtx.lineWidth = 1;
  selCtx.setLineDash([4, 4]);
  selCtx.globalCompositeOperation = 'difference';  // 反色
  selCtx.strokeStyle = '#ffffff';
  selCtx.lineDashOffset = antsDashOffset;
  selCtx.stroke(_antsPath);
  selCtx.globalCompositeOperation = 'source-over';  // 重置
  antsDashOffset = (antsDashOffset - 1 + 8) % 8;
  antsRafId = requestAnimationFrame(drawAnts);
}
```

Marquee 拖拽预览（约第 2428 行）同样删除 `save/scale/restore`，clearRect 改为逻辑坐标。
