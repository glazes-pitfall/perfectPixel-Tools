**Test:** 打开 http://localhost:5010/editor，上传图片，点击「色卡限制」标题展开面板。

#### 2. + 色块交互

**Test:** 生成色卡后，在调色盘选择一种颜色，然后点击色卡末尾的灰色「+」色块。

**Why human:** 动态 DOM 追加和视觉顺序需要浏览器实际运行才能确认。

#### 3. combobox 下拉交互

**Test:** 点击「▼」按钮展开已保存色卡下拉；悬停某条目查看删除按钮；点击「📁 从本地上传」。
**Expected:** 下拉展开时各条目正常显示；悬停时出现红色 ✕；底部有上传选项且点击后弹出文件选择器。
**Why human:** CSS hover 状态和下拉定位（z-index、position:absolute）需要在实际浏览器中验证，特别是 combobox 容器 overflow:hidden 对绝对定位子元素的影响。
