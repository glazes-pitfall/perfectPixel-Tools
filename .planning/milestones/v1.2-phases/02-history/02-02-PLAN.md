**验证步骤：**

**SC-1: 三次点击 → 三次 Undo → 恢复原始状态**

1. `browser_navigate(url="http://localhost:5010/editor")` — 打开编辑器
2. `browser_console_messages(level="error")` — 确认无 JS 错误
3. 读取初始 historyIndex：`browser_evaluate(function="() => EditorState.historyIndex")`
   → 预期：-1（无历史）
4. 读取 Undo 按钮状态：`browser_evaluate(function="() => document.getElementById('btn-undo').disabled")`
   → 预期：true（灰色禁用）
5. 在 canvas 上点击三次（使用 cursor-canvas 的 ref 或坐标，避免 top-bar 区域）
6. 验证 historyIndex = 2（三次 pushHistory 后 index 从 -1→0→1→2）
   → `browser_evaluate(function="() => EditorState.historyIndex")`
   第一次 push: history=[snap0], index=0；第二次: index=1；第三次: index=2
7. 验证 Undo 按钮已启用：`browser_evaluate(function="() => document.getElementById('btn-undo').disabled")`
   → 预期：false
8. 按 Cmd+Z 三次（通过 browser_evaluate 直接调用 undo() 三次）
9. 批量验证最终状态：
   ```javascript
   () => ({
     historyIndex: EditorState.historyIndex,
     historyLength: EditorState.history.length,
     undoDisabled: document.getElementById('btn-undo').disabled
   })
   ```
   预期：historyIndex=0, historyLength=3, undoDisabled=true

   验证逻辑：第一次点击 pushHistory() 保存原始图像 → history[0]；
   三次点击后 index=2；三次 undo 后：index=2→1→0，第三次 undo 因 index<=0 停止不变。

**SC-2: Redo 有效**

在 SC-1 完成后（index=0）：
1. 通过 browser_evaluate 调用 redo() 一次 → index=1
2. 验证 historyIndex=1，Redo 按钮仍启用，Undo 按钮也启用
3. 画布应显示纯色（非原始图像），证明 redo 生效

**SC-3: 按钮常驻顶栏且状态正确**

`browser_snapshot()` — 确认 btn-undo 和 btn-redo 在 top-bar 中可见

**SC-4: history stack 长度等于点击次数**

在全新加载后点击 3 次，验证：
`browser_evaluate(function="() => EditorState.history.length + ':' + EditorState.historyIndex")`
→ 预期：`"3:2"`
